# look_share
패션 스타일 이미지와 글을 공유하고 게시판에서 소통하며 스타일을 공유하는 앱입니다.

주요기능💡
게시물 올리기 : 이미지와 글을 작성하여 게시물을 올리는 기능
게시판 : 유저들이 글을 올려 소통 할 수 있는 기능
채팅 : 실시간으로 채팅 할 수 있는 기능
홈화면 : 업로드한 게시물을 볼 수 있는 기능

코드 실행방법🕹
Frontend : npm run serve
Backend : npm start

제작시 사용한 스킬🛠
Frontend : vue, scss
Backend : node.js, express
DB : mongoDB
